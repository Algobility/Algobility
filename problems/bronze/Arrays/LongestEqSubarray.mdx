# Longest Subarray of Equal Elements  

Given an array of <Math inline>N</Math> integers, find the length of the longest contiguous subarray in which all elements are equal.  

# Input  

- The first line contains an integer <Math inline>N</Math> (<Math inline>1 \leq N \leq 10^5</Math>), the number of elements in the array.  
- The second line contains <Math inline>N</Math> space-separated integers <Math inline>A[1], A[2], ..., A[N]</Math> (<Math inline>-10^9 \leq A[i] \leq 10^9</Math>).  

# Output  

Print a single integer: the length of the longest contiguous subarray where all elements are the same.  

# Example  

Input:  
```
7  
4 4 4 2 2 2 2  
```  

Output:  
```
4
```  

The longest contiguous segment with the same value is <Math inline>[2, 2, 2, 2]</Math>, which has length <Math inline>4</Math>.  

# Solution  

<Solution>  

## Approach  

1. Maintain a variable to track the longest contiguous segment of identical elements.  
2. Iterate through the array while keeping track of the current segment length.  
3. If the current element is the same as the previous one, increase the segment length; otherwise, reset the segment length.  
4. Keep track of the maximum segment length encountered.  

## Implementation  

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int N;
    cin >> N;
    
    int A[N];
    for (int i = 0; i < N; i++) {
        cin >> A[i];
    }

    int maxLen = 1, curLen = 1;

    for (int i = 1; i < N; i++) {
        if (A[i] == A[i - 1]) {
            curLen++;
        } else {
            if (curLen > maxLen) {
                maxLen = curLen;
            }
            curLen = 1;
        }
    }

    if (curLen > maxLen) {
        maxLen = curLen;
    }

    cout << maxLen << endl;
}
```

## Complexity Analysis  

Time complexity is <Math inline>O(N)</Math> since we iterate through the array once.  
Space complexity is <Math inline>O(1)</Math> as only a few integer variables are used.
