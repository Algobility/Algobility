Sorting is the process of taking an array and rearranging the elements such that they are ordered in ascending or descending order. C++ provides us with a built-in function that can sort an array. # Sorting a simple array

To sort an array, use `sort(arr, arr + n)` where `n` is the number of elements to be sorted. By default, the `sort()` function sorts an array in ascending order. 

Note: `sort()` modifies the array passed in instead of simply returning a sorted version of it.

```cpp
#include <bits/stdc++.h>    
using namespace std;

int main() {
    int arr[] = { 2,4,2,5,2,3 };
    sort(arr, arr + 6);
    for (int i = 0; i < 6; i++) {
        cout << arr[i] << ' ';
    }
}
```

The following code outputs

```
2 2 2 3 4 5 
```

To solve this problem, we can simply sort the array in ascending order and then print out the last 3 elements of the modified array.

```cpp
#include <bits/stdc++.h>    
using namespace std;

int main() {
    int n; cin >> n;
    int arr[n];
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    sort(arr, arr + n);
    for (int i = n - 3; i < n; i++) {
        cout << arr[i] << ' ';
    }
}
```

# Sorting vectors
Sorting vectors isn't much different. Simply use `sort(v.begin(), v.end())` (or `sort(begin(v),end(v))`). The default sort function sorts the array in ascending order. 

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
	vector<int> v{1, 3, 3, 120, 4};
	sort(v.begin(), v.end());
    //v is now sorted
}
```
